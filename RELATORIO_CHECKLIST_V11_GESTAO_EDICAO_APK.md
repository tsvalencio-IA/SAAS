# Checklist V11 — Gestão, edição, exclusão e APK separado

Alterações:

- Gestor/gerente/admin podem excluir checklist salvo.
- Gestor/gerente/admin podem abrir checklist para edição.
- Gestor/gerente/admin podem adicionar item manual em qualquer seção.
- Gestor/gerente/admin podem editar a descrição de item do checklist.
- Itens técnicos adicionados:
  - retífica de discos;
  - retífica de tambores;
  - retífica de campanas;
  - troca de lona/sapata;
  - regulagem de alavanca/freio de mão;
  - regulagem de portas, fechaduras e batentes;
  - regulagem/conferência de válvulas;
  - regulagem de trambulador/câmbio.
- Manifest próprio do checklist ajustado.
- Incluída pasta `checklist-apk` com projeto Capacitor para gerar APK separado.

Observação técnica:
O link `/checklist.html` pode abrir no app principal se o PWA principal já estiver instalado com escopo do domínio inteiro. Para ícone separado 100% independente, use o APK separado desta versão ou um subdomínio exclusivo.
