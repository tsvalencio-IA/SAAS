# APK separado do Checklist OFICIN-IA — V12

Esta versão usa packageId único:

`br.com.thiaguinhosolucoes.oficinia.checklistsolo.v12`

Por isso instala separado do APK principal.

Se ainda aparecer “já instalado”, o APK gerado não é desta pasta `checklist-apk` ou o celular está recebendo APK antigo. Gere novamente pelo workflow desta pasta, baixe o artefato novo e instale.

O app principal instalado pode capturar links do mesmo domínio no navegador/PWA. Para abrir separado, use o ícone do APK Checklist Solo instalado.
