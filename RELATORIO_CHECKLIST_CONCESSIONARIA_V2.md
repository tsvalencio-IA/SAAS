# Checklist Padrão Concessionária OFICIN-IA v2

Arquivos novos/alterados:
- checklist.html
- js/checklist.js
- data/checklist-model.json
- service-worker.js
- cópias em capacitor-android/www

O que foi adicionado:
- UX mobile first, sem rolagem lateral.
- Tema claro/escuro.
- Desenhos/ícones por sistema do veículo.
- Itens completos de concessionária: lâmpadas, borrachas, fechaduras, botões, acessórios, palhetas, freios, suspensão, vazamentos, fluidos, arrefecimento, elétrica, motor, pneus, transmissão, interior e teste de rodagem.
- Histórico por placa com última ocorrência semelhante por item.
- Sugestões inteligentes de conjunto, como freio + disco + fluido e amortecedor + kit + batente + coifa.
- Comentário por item.
- Fotos por item e fotos gerais.
- Gravação de áudio e ditado de texto quando suportado pelo navegador.
- PDF com fotos anexadas.
- XLSX para importação/conferência.
- JSON para anexar/importar na O.S.
- Salvamento local de rascunho no celular.

Observação técnica:
- Não altera Jarvis, OS, financeiro, estoque, NF/XML, cliente ou equipe.
- É um módulo separado para ser integrado depois via botão no Jarvis.
